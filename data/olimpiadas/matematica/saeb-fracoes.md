# SAEB - Frações e Operações

## Questão Típica SAEB

**Pedro comeu 2/5 de uma pizza e Maria comeu 1/4 da mesma pizza. Que fração da pizza foi consumida ao todo?**

a) 3/9  
b) 3/20  
c) 13/20  
d) 2/9

---

## Resolução Detalhada

### Passo 1: Entender o Problema

- Pedro comeu: **2/5** da pizza
- Maria comeu: **1/4** da pizza
- Queremos saber: **Total consumido = 2/5 + 1/4**

### Passo 2: Somar Frações com Denominadores Diferentes

Para somar frações, precisamos de um **denominador comum**.

**Denominadores:** 5 e 4  
**MMC(5, 4) = 20**

### Passo 3: Transformar as Frações

**Converter 2/5 para denominador 20:**
```
2/5 = (2 × 4)/(5 × 4) = 8/20
```

**Converter 1/4 para denominador 20:**
```
1/4 = (1 × 5)/(4 × 5) = 5/20
```

### Passo 4: Somar

```
8/20 + 5/20 = (8 + 5)/20 = 13/20
```

---

## Resposta Final

✅ **Alternativa C: 13/20**

Pedro e Maria consumiram **13/20** (ou 65%) da pizza.

---

## Conceitos SAEB - Frações

### 1. MMC (Mínimo Múltiplo Comum)

Para somar frações com denominadores diferentes:
1. Calcule o MMC dos denominadores
2. Transforme ambas as frações para o MMC
3. Some os numeradores
4. Mantenha o denominador

### 2. Equivalência de Frações

```
2/5 = 4/10 = 8/20 = ...
```

Multiplicando numerador e denominador pelo mesmo número, a fração não muda!

### 3. Simplificação

Sempre verifique se a resposta final pode ser simplificada:
```
13/20 → Não pode simplificar (13 é primo)
```

## Outras Operações com Frações

### Multiplicação
```
2/3 × 4/5 = (2 × 4)/(3 × 5) = 8/15
```
Multiplica numerador com numerador, denominador com denominador.

### Divisão
```
2/3 ÷ 4/5 = 2/3 × 5/4 = 10/12 = 5/6
```
Inverte a segunda fração e multiplica.
