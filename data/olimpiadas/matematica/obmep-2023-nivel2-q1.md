# OBMEP 2023 - Nível 2 - Questão 1

## Enunciado

Ana comprou 5 cadernos por R$ 12,00 cada um e 3 canetas por R$ 2,50 cada. Ela pagou com uma nota de R$ 100,00.

**Quanto Ana recebeu de troco?**

a) R$ 27,50  
b) R$ 30,00  
c) R$ 32,50  
d) R$ 35,00  
e) R$ 37,50

---

## Resolução Passo a Passo

### Passo 1: Calcular o gasto com cadernos
```
5 cadernos × R$ 12,00 = R$ 60,00
```

### Passo 2: Calcular o gasto com canetas
```
3 canetas × R$ 2,50 = R$ 7,50
```

### Passo 3: Calcular o gasto total
```
R$ 60,00 + R$ 7,50 = R$ 67,50
```

### Passo 4: Calcular o troco
```
R$ 100,00 - R$ 67,50 = R$ 32,50
```

---

## Resposta Final

✅ **Alternativa C: R$ 32,50**
