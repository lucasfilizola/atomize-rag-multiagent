# Função Quadrática

## O que é uma Função Quadrática?

Uma função quadrática é uma função polinomial de segundo grau, que pode ser escrita na forma:

**f(x) = ax² + bx + c**

Onde:
- **a, b e c** são números reais
- **a ≠ 0** (se a = 0, a função deixa de ser quadrática)

## Características Principais

### 1. Gráfico (Parábola)
O gráfico de uma função quadrática é sempre uma **parábola**.

- Se **a > 0**: parábola com concavidade para cima (formato de "U")
- Se **a < 0**: parábola com concavidade para baixo (formato de "∩")

### 2. Vértice
O vértice é o ponto máximo ou mínimo da parábola.

Coordenadas do vértice:
- **xᵥ = -b / (2a)**
- **yᵥ = -Δ / (4a)** onde Δ = b² - 4ac

### 3. Raízes (Zeros da Função)
São os valores de x onde f(x) = 0.

Fórmula de Bhaskara:
**x = (-b ± √Δ) / (2a)**

Onde Δ = b² - 4ac (discriminante)

- Se Δ > 0: duas raízes reais distintas
- Se Δ = 0: uma raiz real (raiz dupla)
- Se Δ < 0: não há raízes reais

## Exemplos

### Exemplo 1:
f(x) = x² - 4x + 3

- a = 1, b = -4, c = 3
- Δ = (-4)² - 4(1)(3) = 16 - 12 = 4
- Raízes: x = (4 ± 2) / 2 → x₁ = 3, x₂ = 1
- Vértice: xᵥ = 4/2 = 2, yᵥ = -4/4 = -1
- Ponto mínimo: (2, -1)

### Exemplo 2:
f(x) = -x² + 2x + 3

- a = -1, b = 2, c = 3
- Δ = 4 - 4(-1)(3) = 4 + 12 = 16
- Raízes: x = (-2 ± 4) / (-2) → x₁ = -1, x₂ = 3
- Vértice: xᵥ = -2/(-2) = 1, yᵥ = -16/(-4) = 4
- Ponto máximo: (1, 4)

## Aplicações nas Olimpíadas

Questões de olimpíadas frequentemente envolvem:
1. Encontrar máximos e mínimos em problemas de otimização
2. Análise de movimento de projéteis (física)
3. Problemas de geometria analítica
4. Inequações quadráticas

## Dica para Olimpíadas

**Sempre identifique:**
- O sinal de "a" (concavidade)
- O discriminante Δ (quantidade de raízes)
- O vértice (ponto crítico)
- O eixo de simetria (x = -b/2a)
